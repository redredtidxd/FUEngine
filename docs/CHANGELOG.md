# Changelog — FUEngine

Historial orientado a usuarios y desarrolladores. La versión canónica del ejecutable está en `FUEngine.Core` (`EngineVersion.Current`). Desde el Hub puedes abrir este archivo si existe junto al repositorio (enlace «CHANGELOG»).

## 0.0.1

- Editor WPF: mapa por capas y chunks, escenas múltiples, proyectos `.FUE`, Lua (NLua), Play embebido, exportación.
- Hub de inicio: proyectos recientes, biblioteca global, plantillas, configuración del motor.
- Documentación in-app (`EngineDocumentation`) integrada en el ejecutable.
