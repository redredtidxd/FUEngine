# Documentación técnica (FUEngine)

La **fuente de verdad** para clonar, compilar, instalador y flujo de desarrollo es el **[README.md](../README.md)** en la raíz del repositorio.

Este directorio contiene material de referencia para mantenedores e IAs (p. ej. **`AI-ONBOARDING.md`**). No dupliques aquí procedimientos de publicación: si cambian, actualízalos solo en el README de la raíz y en la ayuda integrada del editor (`FUEngine/Help/`) cuando afecte a usuarios.
